Bienvenue sur l'API de vérification des numéros de téléphone !

Cette API vous permet de récupérer des informations détaillées sur un numéro de téléphone, telles que :

  • Le pays d'origine
  • L'opérateur de télécommunication
  • Les coordonnées géographiques (latitude et longitude)
  • La validité du numéro
  • Une carte HTML de la localisation

Comment utiliser l'API :

  • Méthode GET : Envoyez une requête avec le paramètre ?phone_number= (ex. ?phone_number=+918897909596)
  • Méthode POST : Envoyez un corps JSON avec {"phone_number": "valeur"} (ex. {"phone_number": "+918897909596"})

Exemple de réponse JSON :

    {{
        "phone_number": "+918897909596",
        "country_code": 91,
        "location": "India",
        "service_provider": "Airtel",
        "latitude": 28.6139,
        "longitude": 77.2090,
        "is_valid_number": true,
        "number_type": "MOBILE",
        "map_html": "..."
    }}

L'heure actuelle est {{ current_time }} GMT.
